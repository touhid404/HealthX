# HealthX

HealthX is a comprehensive healthcare management platform designed to streamline operations for clinics, hospitals, and healthcare professionals. It provides robust features for appointment scheduling, patient management, medical reporting, billing, and more.

## Features

- User authentication and authorization
- Admin, doctor, and patient modules
- Appointment scheduling and management
- Medical reports and prescriptions
- Payment processing (Stripe integration)
- Review and feedback system
- Cloudinary integration for file uploads
- Email notifications

## Tech Stack

- **Backend**: Node.js, TypeScript, Express.js
- **Database**: PostgreSQL (via Prisma ORM)
- **Authentication**: JWT, OAuth (Google)
- **Payments**: Stripe
- **File Uploads**: Cloudinary, Multer
- **Email**: Nodemailer
